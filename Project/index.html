What we provide

This website has been created to provide the user with the necessary tools for managing the curriculum for Reading University, more specifically the Computer Science department.

Create a Module

This tool within the Computer Science Curriculum Management System allows for users to create new modules in the curriculum through the use of an easy-to-use form. Users must be logged in to access this webpage.

Module Portfolio

This webpage will allow you to view all the modules, or filter by year, category and topic which will allow for comparison of modules, or to see which modules teach your favourite topic.

Edit Modules

This system also provides a tool for edit modules, for instance, the module code or module title. Users must be logged in to access this webpage.
